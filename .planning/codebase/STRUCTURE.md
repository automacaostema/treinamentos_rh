# STRUCTURE.md

## Estrutura de Arquivos
- `.planning/`: Documentação de desenvolvimento e processos.
- `index.html`: Ponto de entrada e estrutura da UI.
- `server.js`: API Backend Express para conexão com MariaDB.
- `package.json`: Gerenciamento de dependências do Node.js.
- `styles.css`: Estilização global e componentes UI.
- `app.js`: Lógica do frontend consumindo API local.
- `pdf-service.js`: Módulo isolado para geração de PDFs.
- `config.json`: Endpoint da API local.
- `logo.png`: Asset visual da empresa.
